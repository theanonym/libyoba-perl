Detailed Description

Hash reference

Example
Hash hash = perl.newHash<IV>({ { "a", 1 }, { "b", 2} }); // (my) %hash = (a => 1, b => 2)
hash.insert("c", perl.newScalar(3)); // $hash{c} = 3
Scalar a = hash.get("a").makeCopy(); // $a = $hash{a}
