Detailed Description

Scalar reference

Example
Scalar scalar = perl.newScalar("str"); // (my) $scalar = "str"
scalar.setValue(14); // $scalar = 14
Scalar copy = scalar.makeCopy(); // $copy = $scalar
Scalar ref = scalar.makeRef(); // $ref = \$scalar
